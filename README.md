🎯 SENSIX RANZ - Premium Game Injector 🚀
SENSIX RANZ adalah aplikasi injector game (Free Fire) berbasis web modern yang dirancang untuk memberikan pengalaman bermain yang lebih halus, presisi, dan optimal. Dengan antarmuka Neon Futuristic UI, aplikasi ini menggabungkan keindahan visual dengan fungsionalitas tingkat tinggi. 🛠️✨
🌟 Fitur Unggulan (Main Features)
Berikut adalah daftar fitur yang tersedia di dalam Sensix Ranz:
🎯 Combat Features
 * Aim Lock 5% 🎯: Mengunci bidikan ke arah musuh dengan bantuan akurasi tambahan agar peluru tidak melenceng.
 * Aim Body 🛡️: Memaksa koordinat bidikan untuk tetap berada di area badan musuh.
 * Aim Kepala 💀: Meningkatkan prioritas crosshair ke arah kepala musuh untuk headshot lebih mudah.
 * Drag Smooth 🌪️: Memberikan efek tarikan tombol tembak yang lebih halus dan konsisten.
⚙️ Optimization & System
 * Pelicin Layar 🧴: Meningkatkan sensitivitas sentuhan (touch response) agar movement lebih lincah.
 * FPS Boost 🚀: Mengoptimalkan pemrosesan sistem untuk menjaga kestabilan frame rate tinggi.
 * DPI 500 🖱️: Menyesuaikan kepadatan pixel untuk kontrol kursor yang lebih presisi.
 * Game Mode API 🌐: Menggunakan jalur data khusus untuk meminimalisir ping dan delay jaringan.
🎨 Visual & Experience
 * Grafik Minecraft 🧱: Mengubah tekstur menjadi ultra-low untuk performa maksimal di HP spesifikasi rendah.
 * Neon Aesthetic UI 🌈: Tampilan gelap dengan aksen neon yang nyaman di mata untuk penggunaan lama.
 * Notification System 🔔: Notifikasi status (ON/OFF) di tengah layar yang disertai suara "Bib.." sebagai indikator.
🚀 Cara Menggunakan (How to Use)
 * Halaman Pertama (Create Account) 📝
   Jika belum punya akses, silakan daftar menggunakan Email, Username, dan Password di menu pendaftaran.
 * Halaman Kedua (Login) 🔑
   Masukkan akun yang sudah dibuat (atau gunakan akun default admin).
 * Halaman Ketiga (Dashboard) 🎮
   Pilih fitur yang ingin diaktifkan. Tunggu jeda aktivasi (1-2 detik) sampai terdengar suara konfirmasi.
